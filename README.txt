Для начала скачать Node и npm
Версии, которые стоят у меня:
    - Node v18.16.0
    - npm v9.5.1
Открыть терминал и ввести комманды:
    1 - npm i
    2 - npm run dev
ВАЖНО!! Ввойти в аккаунт и включить только:
    1 - Получать уведомления о входящих сообщениях и файлах(ДА)
    2 - Интервал отправки сообщений из очереди(500 мсек)
